PROJECTS SPOTLIGHT

  • Kisii Land Use and Land Cover Analysis

    Personal Project

    This project aims to analyze and map the Land Use and Land Cover (LULC) of Kisii county in Kenya, East Africa. The analysis utilizes ESRI‚Äôs ArcGIS Pro and QGIS software to classify land cover types, such as built-up areas, agricultural land, and forests, as of 2024. The results will support sustainable land management, environmental management, and urban planning.

  • Kenya's 2019 Population Density

    Personal Project

    In this project, Kenya's 2019 population census is analyzed using Python's Pandas, Geopandas. The results are presented in an interactive choropleth map using leaflet.js. Each county's population density can be viewed on the web map by clicking on the specific county.

  • Cape Town 2017 Water Consumption

    Personal Project

    This project features the 2017 water crisis in Cape Town city in South Africa using pandas and folium libraries. The web maps show the 2017 average water Consumption per suburb in the month of October. Users can select points showing pop-up of the suburb name and water Consumption per kilolitre.

  • Kenya Population Density Analysis

    Personal Project

    This project examines Kenya's 2019 population census using Python's Pandas, Geopandas, and Matplotlib. The results are presented in tables, bar graph, choropleth map, and interactive map. Each county's population density can be viewed on the interactive map upon hovering on the specific county.
